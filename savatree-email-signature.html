📋 How to Install in Gmail or Outlook
1
Click "Copy Signature" button above — this copies the full HTML signature to your clipboard.
2
Gmail: Go to Settings (⚙️) → See all settings → Signature tab → Create new signature → Paste (Ctrl+V / Cmd+V) into the signature box.
3
Outlook: File → Options → Mail → Signatures → New → Paste into the signature editor. Switch editor to HTML mode if needed.
4
Update your direct cell number in the signature before installing — click "Edit Fields" to make it editable.
